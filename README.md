# 鸭口选股 V4

> 在 V3 六大指标基础上，新增「当前日线/周线/月线同时处于 BOLL 鸭口扩张」强过滤条件。

## 策略说明

### V4 新增条件

**当前三周期鸭口（强过滤）**：要求最新一根 K 线，日线、周线、月线**当前**均处于 BOLL 开口扩张（鸭口）状态：
- 日线当前：UB↑ & MID↑ & LB↓
- 周线当前：UB↑ & MID↑ & LB↓
- 月线当前：UB↑ & MID↑ & LB↓

### 原有六大指标（V3 全部保留，条件不变）

| 指标 | 月线 | 周线 | 日线 |
|------|------|------|------|
| BOLL | 12 个月内出现 UB↑&BOLL↑&LB↓ | 26 周内出现 | 22 日内出现 |
| MACD | 12 个月内金叉后 DIF 持续≥DEA | 26 周内零轴上方金叉持续 | 22 日内金叉后持续 |
| OBV  | OBV > MA(OBV,20) | 同 | 同 |
| DMA  | DIF\_DMA > DIFMA | 同 | — |
| AMO  | 52 周内任意周 vol>前周 3 倍 & 26 周内>1.5 倍 & 22 日内>1.5 倍 | | |
| KDJ  | 24 个月内 J穿K穿D 三线金叉 | 26 周内 | 22 日内 |

六大指标**全部满足**才入选。

## 运行方式

```bash
pip install -r requirements.txt
python strategy.py
```

结果输出到 `docs/index.html`（GitHub Pages 展示）和 `docs/data.json`。

## 自动运行

通过 GitHub Actions，每个交易日（周一至周五）北京时间 16:30 自动运行，结果发布到 GitHub Pages。

## 数据来源

腾讯财经前复权 K 线接口（免费公开接口）。
